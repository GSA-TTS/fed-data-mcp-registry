# Open Government Data MCP Servers

A curated collection of Model Context Protocol (MCP) servers providing programmatic access to open government datasets.

## Available Servers

| Dataset | Agency | GitHub URL | Remote URL |
|---------|--------|------------|------------|
| U.S. Census | U.S. Census Bureau | [github.com/uscensusbureau/us-census-bureau-data-api-mcp](https://github.com/uscensusbureau/us-census-bureau-data-api-mcp) | |

## Contributing

Have an MCP server for government data? Submit a pull request to add it to the table!

## Usage

To use these servers, add the Remote URL to your MCP client configuration.
